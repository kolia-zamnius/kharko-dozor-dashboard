---
title: UI map
---

The dashboard has a thin top navbar with two main routes:

- **Users** (`/users`) — every visitor whose session has been linked via `dozor.identify()`. Anonymous sessions don't appear here.
- **Replays** (`/replays`) — the full session list, identified or not.

Plus the avatar dropdown on the right, which exposes:

- **Switch organisation** — sub-menu listing every org you're a member of.
- **Manage organisations** — settings hub for orgs + projects + members.
- **User settings** — your profile, locale, passkeys, etc.
- **Documentation** — this site.
- **Light / Dark mode** — theme toggle.
- **Sign out**.

Use the sidebar to navigate to specific surfaces.
